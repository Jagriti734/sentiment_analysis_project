# Sentiment Analysis Project

## Overview

This project is a Natural Language Processing (NLP) and Machine Learning application that classifies text sentiment as Positive or Negative. The system analyzes user-generated text such as movie reviews, comments, customer feedback, and social media posts to determine the overall sentiment.

The application utilizes text preprocessing, TF-IDF vectorization, and machine learning techniques to perform real-time sentiment prediction through an interactive Streamlit web interface.

## Features

* Natural Language Processing (NLP)
* Text Cleaning and Preprocessing
* TF-IDF Feature Extraction
* Sentiment Classification
* Real-Time Prediction
* Interactive Streamlit Interface
* Cloud Deployment

## Tech Stack

* Python
* Natural Language Processing (NLP)
* Scikit-learn
* Pandas
* NumPy
* Streamlit
* TF-IDF Vectorization
* Pickle

## Applications

* Movie Review Analysis
* Social Media Sentiment Analysis
* Customer Feedback Analysis
* Product Review Analysis
* Opinion Mining

## Performance

* Accuracy: ~89%
* Classification: Positive / Negative
